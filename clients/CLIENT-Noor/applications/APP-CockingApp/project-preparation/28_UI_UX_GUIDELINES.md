## 10. الصور والوسائط

| العنصر | المواصفات |
|--------|-----------|
| صورة رئيسية لوصفة | عرض كامل (100%)، ارتفاع متناسب، زوايا `12px` |
| معرض صور | Grid 2-3 أعمدة، زوايا `10px` |
| صور مصغرة | `150x150px`، زوايا `8px` |
| فيديو YouTube | Embed بنسبة 16:9، زوايا `12px` |
| أيقونات | خطية (Lucide أو Heroicons)، حجم `20px` افتراضي |

---

## 11. الاستجابة (Responsive)

| الشاشة | العرض | تعديلات |
|--------|-------|---------|
| Desktop | > 1024px | تخطيط كامل، 3-4 أعمدة للوصفات |
| Tablet | 768-1024px | 2-3 أعمدة، تقليل الهوامش |
| Mobile | < 768px | عمود واحد، قائمة منسدلة للـ Nav، تصغير العناوين |

## 12. الـ Design Gaps

> الفجوات في نظام Claude design التي تحتاج حل أثناء التنفيذ:

| الفجوة | الحل المقترح |
|--------|-------------|
| ألوان الـ hover للروابط في النصوص | استخدام `#b8674e` (أغمق قليلاً من المرجاني) |
| ألوان الـ active/disabled للأزرار | `#e8c9bd` للأزرار المعطلة |
| تدرجات الظل للبطاقات | `0 2px 8px rgba(24,23,21,0.06)` للعادي، `0 4px 16px rgba(24,23,21,0.1)` للرفع |
| مكون الـ Toast/Notification | خلفية حسب الحالة مع أيقونة مناسبة |
| مكون الـ Modal/Dialog | خلفية بيضاء، ظل كبير، زوايا `16px`، خلفية شفافة خلفه |
| حقل البحث مع اقتراحات | قائمة منسدلة بيضاء مع ظل |
