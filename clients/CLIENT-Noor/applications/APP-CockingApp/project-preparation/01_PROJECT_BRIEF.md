## 2. أهداف المشروع

1. **تنظيم الوصفات** — إدارة احترافية مع ربط المكونات ككيانات منفصلة (تجنب الكتابة الحرة)
2. **تجربة مشاهدة ممتعة** — عرض منظم للوصفات مع صور وفيديو وخطوات تحضير
3. **لوحة تحكم كاملة** — للـ Admin إدارة المحتوى بسهولة (وصفات، تصنيفات، مكونات، تعليقات)
4. **تفاعل الزوار** — تعليقات، تحميل PDF، حفظ مفضلة، مقياس ديناميكي
5. **قابلية التوسع** — بنية تقنية تسمح بإضافة المراحل اللاحقة (Phase 2, 3)

---

## 3. نطاق النسخة الأولى (MVP)

### Core 1A — الحد الأدنى للإطلاق ✓
- لوحة تحكم Admin مع إحصائيات
- إدارة كاملة للتصنيفات والمكونات والوصفات
- ربط المكونات بالوصفات مع كميات ووحدات قياس
- رفع صور وروابط فيديو YouTube
- عرض عام للوصفات مع تصفح وتفاصيل
- إدارة التعليقات (موافقة/حذف)

### Extended 1B — قيمة إضافية للنسخة الأولى ✓
- تحميل PDF
- تعليقات عامة (موافقة مسبقة)
- 🛒 قائمة مشتريات ذكية
- 📐 مقياس ديناميكي للكميات
- ⏱️ وقت تحضير وطهي + بحث
- ❤️ مفضلة (تخزين محلي)

## 5. أبرز القرارات

- الاسم **CockingApp** (وليس CookingApp)
- **لا تسجيل دخول** للزوار — تجربة فورية بدون عوائق
- **لا تطبيق جوال** في المرحلة الأولى
- **التعليقات** تحتاج موافقة Admin قبل النشر
- **مكونات ككيانات منفصلة** — مرونة في الاستخدام وإعادة الاستخدام
