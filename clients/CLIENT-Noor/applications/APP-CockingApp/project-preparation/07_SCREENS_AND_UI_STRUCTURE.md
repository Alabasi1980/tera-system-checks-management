## 3. محتوى كل شاشة

### SC-01 — الصفحة الرئيسية `/`

| العنصر | الوصف |
|--------|-------|
| Hero/Banner | (اختياري) ترحيب أو وصفة مميزة |
| قسم "أحدث الوصفات" | بطاقات وصفات مرتبة حسب تاريخ النشر (max 6-8) |
| قسم "تصنيفات" | عرض التصنيفات كبطاقات |
| قسم "وصفات سريعة" (1B) | وصفات وقت تحضير ≤ 15 دقيقة |

### SC-04 — صفحة وصفة مفصلة `/recipes/[slug]`

| العنصر | الوصف |
|--------|-------|
| صورة رئيسية + معرض صور | |
| عنوان الوصفة | مع أيقونة المشاركة |
| ⏱️ وقت التحضير والطهي (1B) | |
| 📐 مقياس ديناميكي (1B) | أداة تغيير عدد الأشخاص |
| قائمة المكونات | مع الكميات والوحدات |
| خطوات التحضير | قائمة مرقمة |
| فيديو YouTube | (إن وجد) |
| ❤️ حفظ في المفضلة (1B) | |
| تحميل PDF (1B) | زر التحميل |
| قسم التعليقات (1B) | عرض + إضافة تعليق |
| وصفات مقترحة | وصفات من نفس التصنيف |

### SC-09 — Admin Dashboard `/admin`

| العنصر | الوصف |
|--------|-------|
| بطاقة "إجمالي الوصفات" | عدد + change |
| بطاقة "التصنيفات" | عدد |
| بطاقة "المكونات" | عدد |
| بطاقة "التعليقات" | عدد المعلقين + عدد المعلّقة |

| جدول "أحدث التعليقات" (1B) | آخر التعليقات غير المعتمدة |

### SC-11 — إنشاء وصفة `/admin/recipes/new`

| العنصر | الوصف |
|--------|-------|
| حقل عنوان الوصفة | نص |
| اختيار التصنيف | قائمة منسدلة (مطلوب) |
| وصف قصير | نص (اختياري) |
| **المكونات** | |
| - اختيار مكون | Autocomplete/search |
| - الكمية | رقم (عشري) |

| - عرض المكونات المضافة مع إمكانية الحذف | |

| - حقل نص الخطوة | Textarea |
| - زر "+ إضافة خطوة" | |
| - ترتيب بالـ drag or up/down | |
| **الصور** | |
| - رفع صور متعددة | File upload |
| - اختيار الصورة الرئيسية | |
| **الفيديو** | |
| - رابط YouTube | نص (اختياري) |

| - وقت الطهي | رقم (دقائق) |

## 5. ملاحظات RTL

- الاتجاه: `dir="rtl"` على مستوى `<html>` + `lang="ar"`
- CSS: تجنب الخصائص الاتجاهية الصلبة (`left`, `right`) — استخدام `inset-inline-start/end`
- Tailwind: استخدام `rtl:` prefix عند الحاجة
- الأيقونات: مراعاة الانعكاس التلقائي للأيقونات الاتجاهية
- القوائم: محاذاة النص لليمين
- النماذج: تسميات الحقول على اليمين

## 6. API Routes (Next.js App Router)

### الواجهة العامة

| الطريقة | المسار | الوظيفة |
|--------|--------|---------|
| GET | `/api/recipes` | قائمة الوصفات (بحث، تصفية، ترقيم) |
| GET | `/api/recipes/[slug]` | تفاصيل وصفة |
| GET | `/api/categories` | قائمة التصنيفات |
| GET | `/api/categories/[slug]/recipes` | وصفات حسب التصنيف |
| GET | `/api/recipes/[slug]/comments` | تعليقات وصفة (معتمدة فقط) |
| POST | `/api/recipes/[slug]/comments` | إضافة تعليق (1B) |
| GET | `/api/recipes/[slug]/pdf` | تحميل PDF (1B) |

### لوحة التحكم

| الطريقة | المسار | الوظيفة |
|--------|--------|---------|
| GET | `/api/admin/dashboard` | إحصائيات Dashboard |
| GET/POST | `/api/admin/categories` | قائمة / إضافة تصنيف |
| GET/PUT/DELETE | `/api/admin/categories/[id]` | تعديل / حذف تصنيف |
| GET/POST | `/api/admin/ingredients` | قائمة / إضافة مكون |
| GET/PUT/DELETE | `/api/admin/ingredients/[id]` | تعديل / حذف مكون |
| GET/POST | `/api/admin/recipes` | قائمة / إضافة وصفة |
| GET/PUT/DELETE | `/api/admin/recipes/[id]` | تعديل / حذف وصفة |
| POST | `/api/admin/recipes/[id]/images` | رفع صورة |
| DELETE | `/api/admin/recipes/[id]/images/[imageId]` | حذف صورة |
| GET/PUT | `/api/admin/comments` | قائمة / موافقة/حذف تعليق (1B) |
| POST | `/api/admin/login` | تسجيل دخول |
| POST | `/api/admin/logout` | تسجيل خروج |
