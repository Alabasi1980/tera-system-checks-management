> سيتم إنشاء ملفات TASK-ID مفصلة في Phase 5 — Execution Planning.
> كل TASK-ID سيتضمن: الوصف، معايير القبول، الملفات المستهدفة، الـ sub-agent (إن وجد).

### نموذج TASK-ID

```
# TASK-COD-001: إعداد مشروع Next.js + Prisma

## الوصف
إنشاء مشروع Next.js جديد مع TypeScript، إعداد Prisma مع PostgreSQL،
تثبيت الاعتماديات الأساسية (Tailwind, lucide-react, ...).

## معايير القبول
- [ ] `next dev` يعمل بدون أخطاء
- [ ] Prisma يتصل بقاعدة البيانات
- [ ] `npx prisma db push` يعمل
- [ ] RTL (Arabic) مضبوط في `layout.tsx`
- [ ] `.env.example` يحتوي على المتغيرات المطلوبة

## الملفات المستهدفة
- package.json, tsconfig.json, next.config.ts, tailwind.config.ts
- prisma/schema.prisma
- src/app/layout.tsx
- src/lib/prisma.ts

## التبعيات
- لا توجد
```
