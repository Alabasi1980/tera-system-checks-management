## 1. داخل النطاق — Core MVP (Phase 1A)

> هذه الوظائف أساسية ولا يمكن إطلاق التطبيق بدونها:

| الوظيفة | الوصف | ملاحظات |
|---------|-------|---------|
| **لوحة تحكم Admin** | Dashboard رئيسي مع إحصائيات (عدد الوصفات، التصنيفات، المكونات، التعليقات) | واجهة خاصة بالـ Admin |
| **إدارة التصنيفات** | إضافة، تعديل، حذف، ترتيب تصنيفات الوصفات | يحددها Admin (مقبلات، رئيسية، حلويات...) |
| **إدارة المكونات** | إضافة، تعديل، حذف مكونات ككيانات منفصلة مع وحدات قياس | كوب، ملعقة كبيرة/صغيرة، كغم، غرام، مل، حبة |
| **إدارة الوصفات** | إنشاء، تعديل، حذف وصفات مع ربط المكونات بالكميات | قلب التطبيق |
| **معرض الصور** | رفع وعرض صور لكل وصفة | صور متعددة لكل وصفة |
| **روابط الفيديو** | إضافة روابط YouTube embed لكل وصفة | لا رفع مباشر |
| **خطوات التحضير** | إضافة نصوص منسقة لخطوات تحضير الوصفة | |
| **الصفحة الرئيسية** | عرض الوصفات مصنفة مع صور | للزوار |
| **صفحة التصنيفات** | تصفح حسب الفئة | |
| **صفحة وصفة مفصلة** | عرض مقادير، خطوات، صور، فيديو | |
| **بحث بسيط** | بحث في عناوين الوصفات | |
| **إدارة التعليقات** | موافقة أو حذف التعليقات من لوحة التحكم | |

## 2. داخل النطاق — Extended MVP (Phase 1B)

> هذه الوظائف مهمة وتضيف قيمة كبيرة، لكن يمكن إطلاق التطبيق بدونها مؤقتاً:

| الوظيفة | الوصف | ملاحظات |
|---------|-------|---------|
| **تحميل PDF** | تحميل كل وصفة بصيغة PDF منسقة | |
| **نظام التعليقات العامة** | تعليقات بدون حساب مع موافقة Admin مسبقة | تحتاج فلترة SPAM |
| **🛒 قائمة مشتريات ذكية** | تجميع مكونات وصفة/وصفات في قائمة مشتريات | من مقترحات البحث |
| **📐 مقياس ديناميكي** | تكبير/تصغير كميات الوصفة (مثلاً: تكفي 4 بدل 2) | |
| **⏱️ وقت التحضير والطهي** | إضافة وقت التحضير والطهي للوصفات + بحث/فلترة | |
| **❤️ مفضلة** | حفظ الوصفات المفضلة (تخزين محلي للزوار) | بدون حساب |

## 6. خارج النطاق نهائياً (Out of Scope)

> هذه الوظائف لن تكون جزءاً من التطبيق في أي مرحلة حالية:

| الوظيفة | سبب الاستبعاد |
|---------|---------------|
| تكامل مع أجهزة المطبخ الذكية | خارج احتياجات العميل الأساسية |
| متجر إلكتروني لبيع المنتجات | خارج نطاق تطبيق الوصفات |
| نظام اشتراكات مدفوعة للزوار | الموقع مجاني للجميع |
| استيراد فيديو من TikTok/Instagram | YouTube فقط |
| نظام أدوار متعدد (Admin واحد فقط) | حسب طلب العميل |
| تسجيل دخول للزوار | تجربة بدون عوائق |
| تطبيق جوال في النسخة الأولى | ويب أولاً |

## 7. الحدود التقنية

| المجال | الحدود |
|--------|--------|
| المنصة | Web App — متصفحات حديثة (Chrome, Firefox, Safari, Edge) |
| الأداء | صفحات سريعة — تحسين الصور والتخزين المؤقت |
| الأمان | حماية لوحة التحكم بكلمة مرور — لا SSL معقد (on-premise) |
| التخزين | الصور: تخزين محلي — الفيديو: YouTube embed |
| PDF | إنشاء من المتصفح (client-side) أو من السيرفر |
| التوافق | RTL كامل — متجاوب مع الجوال (Responsive) |

## 8. معايير القبول للنطاق

- [ ] جميع وظائف **Core 1A** تعمل بشكل كامل
- [ ] لوحة التحكم محمية ولا يمكن للزوار الوصول إليها
- [ ] صفحات الوصفات تظهر بشكل صحيح مع الصور والروابط
- [ ] المكونات مرتبطة ككيانات منفصلة وليست نصوصاً حرة
- [ ] التعليقات تدخل قائمة الموافقة ولا تظهر مباشرة
- [ ] تحميل PDF يعمل بشكل صحيح
- [ ] الموقع متجاوب ويعمل على الجوال
