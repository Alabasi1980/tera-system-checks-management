## 1. Admin Dashboard

### 1.1 البطاقات الأساسية (Stat Cards)

| البطاقة | مصدر البيانات | التحديث |
|---------|---------------|---------|
| إجمالي الوصفات | `COUNT(Recipe) WHERE isPublished = true` | لحظي |
| إجمالي التصنيفات | `COUNT(Category)` | لحظي |
| إجمالي المكونات | `COUNT(Ingredient)` | لحظي |
| التعليقات المعلقة (1B) | `COUNT(Comment) WHERE isApproved = false` | لحظي |

### 1.2 الجداول

| الجدول | البيانات | الحجم |
|--------|----------|-------|
| أحدث الوصفات | آخر 5 وصفات (عنوان، تصنيف، حالة، تاريخ) | 5 rows |
| أحدث التعليقات (1B) | آخر 5 تعليقات غير معتمدة (اسم، وصفة، نص مختصر) | 5 rows |

## 3. تقارير خارج Dashboard

| التقرير | الوصف | المرحلة |
|---------|-------|---------|
| قائمة الوصفات كاملة (في صفحة /admin/recipes) | جدول بجميع الوصفات مع بحث وتصفية | Core 1A |
| قائمة التصنيفات | جدول تصنيفات مع عدد الوصفات لكل تصنيف | Core 1A |
| قائمة المكونات | جدول مكونات مع عدد مرات الاستخدام | Core 1A |
| تقرير التعليقات | جدول جميع التعليقات مع الفلترة (معتمد / معلق) | 1B |
