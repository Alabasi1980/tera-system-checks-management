---

## 2. الموجز

تطبيق ويب لعرض وإدارة وصفات الطبخ. يسمح للـ Admin (نور) بإدارة الوصفات والتصنيفات والمكونات (ككيانات منفصلة)، ويعرضها للزوار بشكل عام بدون تسجيل دخول. يشمل صوراً وفيديو (YouTube embed) وتعليقات عامة مع موافقة مسبقة، وتحميل PDF.

---

## 4. المستخدمون والأدوار

| الدور | الوصف | صلاحيات |
|-------|-------|---------|
| **Admin (نور)** | مدير التطبيق | دخول لوحة التحكم، إدارة كاملة (وصفات، تصنيفات، مكونات، تعليقات) |
| **زائر (Public)** | مستخدم عام | تصفح الوصفات، مشاهدة التفاصيل، تعليق (بدون حساب)، تحميل PDF، حفظ مفضلة |

## 5. الميزات مصنفة حسب الأولوية

### Core MVP (Phase 1A) — أساسي لا يمكن إطلاق بدونه
- لوحة تحكم Admin (Dashboard مع إحصائيات)
- إدارة التصنيفات (إضافة/تعديل/حذف/ترتيب)
- إدارة المكونات ككيانات منفصلة مع وحدات قياس (كوب، ملعقة كبيرة/صغيرة، غرام، كغم، مل، حبة)
- إدارة الوصفات (إنشاء/تعديل مع ربط المكونات بالكميات)
- رفع صور لكل وصفة (معرض صور)
- روابط فيديو YouTube لكل وصفة
- خطوات التحضير (نصوص منسقة)
- عرض عام للوصفات (صفحة رئيسية، تصنيفات، بحث بسيط)
- صفحة وصفة مفصلة (مقادير، خطوات، صور، فيديو)
- إدارة التعليقات (موافقة/حذف من لوحة التحكم)

### Extended MVP (Phase 1B) — مهم للقيمة لكن يمكن تأجيله قليلاً
- تحميل PDF للوصفات
- نظام تعليقات عامة (بدون حساب — موافقة مسبقة من Admin)
- 🛒 قائمة مشتريات ذكية
- 📐 مقياس الوصفات الديناميكي (تكبير/تصغير الكميات)
- ⏱️ وقت التحضير والطهي + بحث/فلترة بالوقت
- ❤️ مفضلة / حفظ الوصفات (تخزين محلي للزوار)

### Phase 2 — تحسينات كبيرة
- 📊 معلومات غذائية (سعرات حرارية، مغذيات)
- 📅 خطة وجبات أسبوعية
- 🔍 بحث بالمكونات المتوفرة
- ⭐ نظام تقييم ونجوم
- 🗑️ تقليل هدر الطعام
- 🌐 استيراد وصفات من المواقع (لصق رابط)
- 🔄 وضع عدم الاتصال (Offline)

### Phase 3 — ذكاء اصطناعي
- 🤖 اقتراح وصفات بالذكاء الاصطناعي
- 🔄 بدائل المكونات الذكية
- 💰 ميزانية التسوق

### Later / Enterprise
- تطبيق جوال (Native / PWA متقدم)
- لائحة مشتريات متعاونة
- تحكم صوتي
- تكامل واتساب

### خارج النطاق
- تكامل أجهزة المطبخ الذكية
- متجر إلكتروني
- اشتراكات مدفوعة
- استيراد فيديو من TikTok/Instagram
- أدوار متعددة

## 6. سير العمل الرئيسي

1. **Admin** ← دخول لوحة التحكم ← إدارة التصنيفات والمكونات
2. **Admin** ← إنشاء/تعديل وصفة ← اختيار تصنيف ← إضافة مكونات بكميات ← خطوات ← صور ← فيديو
3. **زائر** ← تصفح الوصفات (رئيسية، تصنيفات، بحث)
4. **زائر** ← فتح وصفة ← مشاهدة مقادير، خطوات، صور، فيديو
5. **زائر** ← تعليق (ينتظر الموافقة)
6. **Admin** ← إدارة التعليقات (موافقة/حذف)
7. **زائر** ← تحميل PDF / حفظ مفضلة / مقياس ديناميكي

## 7. القيود والافتراضات

| القيد/الافتراض | التفاصيل |
|----------------|----------|
| الاستضافة | On-premise — تفاصيل السيرفر (OS, specs) لم تحدد بعد |
| Auth | لوحة تحكم Admin بكلمة مرور — لا system أدوار متعددة |
| SSL | لا يحتاج SSL معقد (on-premise) |
| التعليقات | عامة بدون حساب، تحتاج موافقة Admin مسبقة |
| التصنيفات | يحددها Admin لاحقاً (مقبلات، رئيسية، حلويات...) |
| وحدات القياس | كوب، ملعقة كبيرة، ملعقة صغيرة، كغم، غرام، مل، حبة |
| الفيديو | روابط YouTube embed فقط |
| الصور | تخزين محلي (على السيرفر) |
