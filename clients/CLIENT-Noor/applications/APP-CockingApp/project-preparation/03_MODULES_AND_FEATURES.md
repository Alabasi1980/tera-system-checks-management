## 2. تفاصيل الموديولات

### M1 — الواجهة العامة (Public)

| الميزة | المعرف | المرحلة | الوصف |
|--------|--------|---------|-------|
| الصفحة الرئيسية | F1.1 | Core 1A | عرض الوصفات مع صور وتصنيفات، ترتيب حسب الأحدث |
| صفحة التصنيفات | F1.2 | Core 1A | عرض الوصفات مصنفة حسب الفئة |
| صفحة وصفة مفصلة | F1.3 | Core 1A | مقادير، خطوات، صور، فيديو، وقت التحضير |
| بحث | F1.4 | Core 1A | بحث في عناوين الوصفات + فلترة بالتصنيف |
| قائمة مشتريات 🛒 | F1.5 | 1B | تجميع مكونات وصفة/وصفات في قائمة قابلة للطباعة |
| مفضلة ❤️ | F1.6 | 1B | حفظ الوصفات المفضلة (LocalStorage) |
| مقياس ديناميكي 📐 | F1.7 | 1B | تكبير/تصغير كميات الوصفة |

### M2 — لوحة تحكم Admin

| الميزة | المعرف | المرحلة | الوصف |
|--------|--------|---------|-------|
| Dashboard | F2.1 | Core 1A | إحصائيات: عدد الوصفات، التصنيفات، المكونات، التعليقات |
| إدارة التصنيفات | F2.2 | Core 1A | CRUD + ترتيب تصنيفات الوصفات |
| إدارة المكونات | F2.3 | Core 1A | CRUD للمكونات مع وحدات القياس |
| إدارة الوصفات | F2.4 | Core 1A | إنشاء/تعديل/حذف وصفات مع ربط المكونات |
| إدارة التعليقات | F2.5 | Core 1A | عرض، موافقة، حذف تعليقات |

### M3 — المحتوى والوسائط

| الميزة | المعرف | المرحلة | الوصف |
|--------|--------|---------|-------|
| معرض الصور | F3.1 | Core 1A | رفع وعرض صور متعددة لكل وصفة |
| روابط فيديو | F3.2 | Core 1A | إضافة روابط YouTube embed |
| خطوات التحضير | F3.3 | Core 1A | نصوص منسقة لخطوات التحضير (قائمة مرقمة) |
| تحميل PDF | F3.4 | 1B | تحميل الوصفة بصيغة PDF منسقة |

### M4 — التفاعل

| الميزة | المعرف | المرحلة | الوصف |
|--------|--------|---------|-------|
| نظام التعليقات | F4.1 | 1B | تعليقات عامة مع موافقة مسبقة من Admin |
| وقت التحضير والطهي | F4.2 | 1B | إضافة وعرض وقت التحضير والطهي |
| مشاركة الوصفة | F4.3 | Core 1A | نسخ رابط الوصفة |

### M5 — البنية التحتية

| الميزة | المعرف | المرحلة | الوصف |
|--------|--------|---------|-------|
| قاعدة البيانات | F5.1 | Core 1A | PostgreSQL + Prisma schema |
| رفع الملفات | F5.2 | Core 1A | رفع وتخزين الصور محلياً |
| Auth لوحة التحكم | F5.3 | Core 1A | حماية لوحة التحكم بكلمة مرور |
| تحسين الأداء | F5.4 | Core 1A | تحسين الصور، تخزين مؤقت، lazy loading |

## 3. خريطة العلاقات بين الموديولات

```
M1 (الواجهة العامة) ← يقرأ من ← M5 (قاعدة البيانات)
M2 (لوحة التحكم)   ← يكتب إلى ← M5 (قاعدة البيانات)
M3 (الوسائط)       ← جزء من ← M2 (يرفع الصور عبر Admin)
                  ← يُعرض في ← M1 (يعرض للزوار)
M4 (التفاعل)       ← مرتبط بـ ← M1 (التعليقات والمشاركة)
```
