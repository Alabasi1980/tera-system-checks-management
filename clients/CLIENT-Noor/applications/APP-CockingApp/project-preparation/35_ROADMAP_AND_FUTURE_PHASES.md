## 1. المراحل الزمنية

### المرحلة 1A — Core MVP (النسخة الأساسية)

| الحالة | ✅ معتمدة |
|--------|-----------|
| **الهدف** | إطلاق تطبيق وصفات متكامل بوظائف أساسية |
| **الميزات** | وصفات، تصنيفات، مكونات (كيانات منفصلة)، صور، فيديو (YouTube)، خطوات، صفحة رئيسية، بحث، Admin Dashboard، Auth |
| **المدة التقديرية** | ~16 ساعة تنفيذ |
| **TASK-IDs** | 001–012 |

### المرحلة 1B — Extended MVP (تحسينات النسخة الأولى)

| الحالة | ✅ معتمدة |
|--------|-----------|
| **الهدف** | إضافة ميزات تفاعلية تزيد من قيمة التطبيق |
| **الميزات** | PDF، تعليقات 🛒قائمة مشتريات، 📐مقياس ديناميكي، ⏱️ وقت + بحث، ❤️مفضلة |
| **المدة التقديرية** | ~8 ساعات تنفيذ |
| **TASK-IDs** | 013–018 |
| **ملاحظة** | يمكن البدء فور انتهاء 1A |

### المرحلة 2 — تحسينات متقدمة

| الحالة | 📅 مؤجلة — خارج نطاق النسخة الأولى |
|--------|-----------|
| **الأولوية** | عالية |
| **الميزات** | 📊 معلومات غذائية، 📅 خطة وجبات، 🔍 بحث بالمكونات، ⭐ تقييم، 🗑️ تقليل هدر الطعام، 🌐 استيراد وصفات |
| **المدة التقديرية** | ~15–20 ساعة |

### المرحلة 3 — ذكاء اصطناعي

| الحالة | 📅 مؤجلة |
|--------|----------|
| **الميزات** | 🤖 اقتراح وصفات بالذكاء الاصطناعي، 🔄 بدائل المكونات، 💰 ميزانية التسوق |
| **المتطلبات** | تكامل مع API ذكاء اصطناعي |

### Later — تطوير مستقبلي

| الحالة | 🔮 استكشافي |
|--------|-------------|
| **الميزات** | تطبيق جوال، لائحة مشتريات متعاونة، تحكم صوتي، تكامل واتساب |

## 2. خارطة التبعيات بين المراحل

```
1A (Core MVP) ──────────────────────────────────→ إطلاق النسخة الأولى
        │
        └── 1B (Extended) ───────→ تحسين النسخة الأولى
                │
                └── Phase 2 ──────→ تحسينات متقدمة
                        │
                        └── Phase 3 ──→ AI Features
                                │
                                └── Later ──→ Future
```
