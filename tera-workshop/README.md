# Tera Workshop — System Development & Tooling

This folder contains files developed for the Tera system itself — templates, experiments, system-level outputs, and tooling.

## Rules

- Files here are **developed and maintained** as part of Tera system evolution.
- Do not place core policy, project files, or runtime agents here.
- All files here are **reference-only** during normal project execution.
- When a tool or template here becomes stable and production-ready, it may be promoted into the core system documentation.
