أنت Tera Agent.

شغّل Tera Self-Diagnosis الكامل (Section 9.1 من tera.md).

أجب على هذه الأسئلة الثمانية بصدق:

```
Tera Self-Diagnosis:

1. ما المرحلة التي أنا فيها؟
   → ...

2. هل أفهم حالة المشروع الحالية؟
   → نعم / لا / جزئياً — السبب: ...

3. هل قرأت PROJECT_STATE.md وTERA_ACTIVE_CONTEXT.md في هذه الجلسة؟
   → نعم / لا — ما قرأته: ...

4. هل أعمل من ملفات رسمية أم من ذاكرة المحادثة؟
   → ...

5. هل يوجد أي تعارض غير محلول مع المستخدم؟
   → نعم / لا — التفاصيل: ...

6. هل افتراضاتي الحالية موثقة في ملفات رسمية؟
   → نعم / لا — الافتراضات غير الموثقة: ...

7. ما أصغر خطوة آمنة يمكنني اتخاذها الآن؟
   → ...

8. هل أحتاج أي توضيح قبل المتابعة؟
   → نعم / لا — ما أحتاجه: ...

النتيجة الإجمالية: CLEAR / UNCLEAR

الإجراء:
- ...
```

قواعد:
- إذا كانت النتيجة UNCLEAR في أي سؤال: أوقف كل تفويض وأخبرني بالسبب المحدد.
- لا تخمّن أو تفترض — إذا لم تعرف، قل ذلك.
- لا تنفذ أي شيء حتى يكون الوضع CLEAR.
