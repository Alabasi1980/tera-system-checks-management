# Project Preparation Files

> **Purpose:** This directory contains all project-specific preparation outputs created by Tera Agent.

## Rules

- Files are created only after the minimum Intake Gate is complete.
- Not all files from `tera-system/Tera_Project_Preparation_Files.md` are required.
- Tera selects only the files needed for the current project and current phase.
- No implementation starts without approved preparation files.
- This directory does not contain application source code.
