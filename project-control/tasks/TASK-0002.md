## Exact Instructions

### 1. تحديث `prisma/schema.prisma`

أضف المحتوى التالي إلى الملف الموجود (احتفظ بـ `generator client` و `datasource db` الموجودين):

#### Enums

```prisma
enum CheckType {
  ISSUED
  INCOMING
}

enum CheckStatus {
  REGISTERED
  IN_HAND
  CASHED
  RETURNED
  CANCELLED
}

enum UserRole {
  ADMIN
  USER
}
```

#### Model: User

```prisma
model User {
  id           Int       @id @default(autoincrement())
  username     String    @unique
  passwordHash String    @map("password_hash")
  displayName  String    @map("display_name")
  role         UserRole
  isActive     Boolean   @default(true) @map("is_active")
  createdAt    DateTime  @default(now()) @map("created_at")
  updatedAt    DateTime  @updatedAt @map("updated_at")
  checks       Check[]

  @@map("users")
}
```

#### Model: Bank

```prisma
model Bank {
  id            Int       @id @default(autoincrement())
  name          String
  branch        String?
  accountNumber String?   @map("account_number")
  notes         String?
  createdAt     DateTime  @default(now()) @map("created_at")
  updatedAt     DateTime  @updatedAt @map("updated_at")
  checks        Check[]

  @@map("banks")
}
```

#### Model: Party

```prisma
model Party {
  id        Int       @id @default(autoincrement())
  name      String
  phone     String?
  notes     String?
  createdAt DateTime  @default(now()) @map("created_at")
  updatedAt DateTime  @updatedAt @map("updated_at")
  checks    Check[]

  @@map("parties")
}
```

#### Model: Check

```prisma
model Check {
  id              Int         @id @default(autoincrement())
  checkNumber     String      @map("check_number")
  type            CheckType
  bankId          Int         @map("bank_id")
  partyId         Int         @map("party_id")
  amount          Decimal     @db.Decimal(12, 2)
  issueDate       DateTime    @map("issue_date") @db.Date
  dueDate         DateTime    @map("due_date") @db.Date
  status          CheckStatus
  statusNote      String?     @map("status_note")
  statusChangedAt DateTime?   @map("status_changed_at")
  notes           String?
  createdById     Int         @map("created_by")
  createdAt       DateTime    @default(now()) @map("created_at")
  updatedAt       DateTime    @updatedAt @map("updated_at")

  bank   Bank  @relation(fields: [bankId], references: [id])
  party  Party @relation(fields: [partyId], references: [id])
  createdBy User @relation(fields: [createdById], references: [id], map: "fk_check_created_by")

  @@map("checks")
}
```

## Acceptance Criteria

1. ✅ `schema.prisma` يحتوي على 4 models: `User`, `Bank`, `Party`, `Check`
2. ✅ الـ enums الثلاثة موجودة: `CheckType`, `CheckStatus`, `UserRole`
3. ✅ العلاقات (FKs) موجودة: Check → Bank, Check → Party, Check → User
4. ✅ `npx prisma generate` يمر بدون أخطاء
5. ✅ الحقول تستخدم أنواع Prisma الصحيحة (Int, String, Decimal, DateTime, Boolean, Enum)
6. ✅ `@@map()` مستخدمة لتعيين أسماء الجداول بأحرف صغيرة (users, banks, parties, checks)
7. ✅ `@map()` مستخدمة لأسماء الحقول بـ snake_case في قاعدة البيانات
8. ✅ لا توجد تغييرات على أي ملف خارج `checks-management/`
9. ✅ `generator client` و `datasource db` لم يتغيرا

## Rollback Plan

2. صحّح الأنواع أو العلاقات
