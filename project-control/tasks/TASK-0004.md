### دفق المصادقة (Authentication Flow)

```
1. زائر → /login ← middleware يسمح (public route)
2. إدخال username + password → Server Action
3. البحث في DB عن user → bcrypt.compare(password, hash)
4. إنشاء JWT: { userId, role, username } → jose.sign()
5. تعيين Cookie: httpOnly, secure (prod), sameSite:lax, maxAge:24h
6. redirect إلى /
7. جميع الـ Requests → middleware:
   - جلب Cookie → jose.verify(token)
   - صالح ← NextResponse.next()
   - غير صالح ← redirect(/login)
   - admin route + non-admin role ← redirect(/)
8. /logout → مسح Cookie → redirect(/login)
```
